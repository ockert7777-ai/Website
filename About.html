My Story

Code, circuits & curiosity

My name is Ockert Oosthuizen, and I grew up fascinated by the machines around me — not just using them, but understanding exactly how they worked. That curiosity turned into a passion, and that passion turned into code.

My programming journey started with Delphi, where I spent 3+ years honing my logic, learning to think algorithmically, and building real desktop applications. Delphi taught me to appreciate clean architecture and efficient code before I ever touched a web browser as a developer.

From there I branched out — Python for automation and scripts, C++ for low-level thinking, and the full HTML / CSS / JavaScript stack for web. I've also dipped into game development, experimenting with game engines and understanding how interactive worlds are constructed.

Away from the screen, I build my own computers from scratch — sourcing components, assembling hardware, troubleshooting — which has given me a deep appreciation for the physical side of IT that most developers never see.
